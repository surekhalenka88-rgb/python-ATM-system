# 🏧 ATM Management System (Python)

## 📌 Project Description
This is a simple ATM Management System developed using Python.  
The program allows users to perform basic ATM operations after entering a correct PIN.

## ✨ Features
- PIN Verification
- Check Account Balance
- Deposit Money
- Withdraw Money
- Exit Transaction
- Continuous Menu using `while True`

## 🛠 Technologies Used
- Python

## 📂 Project Structure
```
ATM.PY
README.md
```

## ▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Open terminal or command prompt
4. Run:

```bash
python ATM.PY
```

```text

### ATM Menu ###

3. Withdraw Amount
4. Exit
```

## 🔄 Workflow
1. Enter ATM PIN
2. Select operation
3. Perform transaction
4. Continue until Exit
